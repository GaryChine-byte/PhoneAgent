# Data 目录说明

## 📁 目录结构

```
data/
├── app_config.json                   # 应用白名单（默认配置，可直接使用）
├── anti_detection_config.json        # 防检测配置（默认配置，可直接使用）
├── shortcuts.json                    # 快捷指令（默认配置，可直接使用）
├── prompt_cards.json                 # 提示词卡片（默认配置，可直接使用）
├── agent.db                  # 客户端数据库（运行时自动生成）
└── screenshots/                      # 截图缓存（运行时自动生成）
```

## ⚙️ 配置文件说明

### 1. `app_config.json` - 应用白名单

控制 Agent 可以操作哪些应用。

**默认配置**：已包含安全的示例配置（系统设置、Termux），可直接使用

**配置项**：
- `package_name`: 应用包名
- `display_name`: 显示名称
- `enabled`: 是否允许操作（`true`/`false`）
- `category`: 分类标签

**安全建议**：
- ✅ 建议开启：系统应用、工具类应用
- ⚠️ 谨慎开启：社交应用（微信、QQ）、支付应用
- ❌ 建议禁止：银行类、证券类应用

### 2. `anti_detection_config.json` - 防检测配置

配置行为随机化策略，让操作更像真人。

**默认配置**：已包含标准的防检测配置，可直接使用

**配置项**：
- `time_randomization`: 操作间随机延迟（0.5-2秒）
- `coordinate_randomization`: 点击坐标随机偏移（±10像素）
- `swipe_humanization`: 贝塞尔曲线滑动轨迹
- `app_whitelist`: 应用访问控制

**推荐设置**：
- 个人使用：可以关闭防检测，提高执行速度
- 批量自动化：必须开启防检测，降低风控风险

### 3. `shortcuts.json` - 快捷指令

预定义的常用操作指令，支持语音唤醒。

**默认配置**：已包含 8 个常用快捷指令，可直接使用

**示例指令**：
- 查看微信消息
- 刷抖音
- 查询违章
- 支付宝扫码
- 查看购物车
- 导航回家
- 播放音乐
- 截图分享

**字段说明**：
- `id`: 唯一标识符
- `title`: 指令名称
- `instruction`: 执行指令（AI 会理解并执行）
- `category`: 分类（社交、娱乐、生活、支付、购物、出行、工具）
- `voice_keywords`: 语音唤醒关键词
- `is_system`: 是否为系统预设

### 4. `prompt_cards.json` - 提示词卡片

AI 执行任务时的行为提示，用于优化执行效果。

**默认配置**：已包含 8 个优化提示卡片，可直接使用

**提示类型**：
- **操作优化**：精确操作、慢速执行
- **应用适配**：微信专用、购物助手
- **安全提示**：防误操作
- **输入优化**：文字输入优化
- **任务规划**：多步任务规划
- **适应性**：界面变化适应

**字段说明**：
- `id`: 唯一标识符
- `title`: 卡片标题
- `description`: 简短描述
- `content`: 详细提示内容（会追加到 AI 的系统提示中）
- `category`: 分类
- `is_system`: 是否为系统预设

## 🗄️ 数据库文件

### `agent.db`

**用途**：Agent服务统一数据库（任务、设备、模型调用）

**字段**：
- 设备 ID、设备名称
- FRP 端口映射
- 最后在线时间

**位置**：`data/agent.db`

## 📸 Screenshots 目录

**用途**：临时存储任务执行过程中的截图

**清理**：
```bash
# 清理截图缓存
rm -rf data/screenshots/*
```
